Simple Pokmon API

Purpose of this api it to get a list of gen 1 pokemon, their sprite, and their types in JSON.

/api/pokemon -- returns all 151 pokemon from gen 1

/api/pokemon/{pokemonname} -- returns the specified pokemons information

'Mewtwo': {
    'type': ['psychic'],
    'sprites':{
        'front': 'red-blue\transparent\150.png',
        'back': 'red-blue\transparent\back\150.png'
    }
},

-------------------------------------------

Object.type = array of that pokemons' types

object.sprites.front = is the their frontal image. (enemy pokemon in game)

object.sprites.front = is the their back image. (your pokemon in game)

/api/weakness -- returns a list of all types and their corresponding weakness

'poison':{
    'weakness':['ground',  'psycic', 'bug']
},
